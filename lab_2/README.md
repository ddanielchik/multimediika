# Multimedia Application
## lab 2

### Описание проекта
Реализовать веб-мультимедиа проект — **«Галерея изображений»** с интерактивными функциями и аудио-визуализацией.

Проект должен работать полностью **локально** на ПК без обращения к интернету.

---

## Функциональные требования

1. **Переключение изображений**
  - Ручное переключение (кнопки "вперед/назад").
  - Автоматическое переключение с настраиваемым пользователем временем (`timing`).

2. **Комментарии к изображениям**
  - Каждое изображение сопровождается текстовым комментарием.

3. **Аудио**
  - Возможность включения и отключения звуковых дорожек.
  - Выбор одной из **трёх или более звуковых дорожек**, либо загрузка собственного аудиофайла.
  - Аудио-визуализация: изменение свойств объектов галереи на основе данных аудио потока.

4. **Ограничения на изображения**
  - Размер изображений не должен превышать **400×300 пикселей**.

---

## Технические требования

- Использовать **HTML5**, **CSS3**, **JavaScript**, **SVG**, **HTML5 Audio API**.
- **Запрещено** использование сторонних библиотек.

### Пример встраивания аудио

```html
<audio src="Kaoma-Lambada.mp3" id="my_audio"></audio>
<script>
  var obj_Audio = document.getElementById("my_audio");
  // или
  // var obj_Audio = new Audio("Kaoma-Lambada.mp3");
